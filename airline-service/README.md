# Airline Service

Микросервис для управления информацией об авиакомпаниях и их операциях.

## Описание

Airline Service отвечает за хранение и управление данными об авиакомпаниях, включая:
- Регистрацию новых авиакомпаний
- Управление информацией о существующих авиакомпаниях
- Получение списка всех авиакомпаний
- Удаление авиакомпаний

## Основные функции

- Создание, чтение, обновление и удаление записей об авиакомпаниях
- Валидация данных авиакомпаний
- Интеграция с другими сервисами через REST API

## Технологии

- Spring Boot
- Spring Data JPA
- PostgreSQL
- Docker

## API Endpoints

- `GET /airlines` - Получение списка всех авиакомпаний
- `GET /airlines/{id}` - Получение информации об авиакомпании по ID
- `POST /airlines` - Создание новой авиакомпании
- `PUT /airlines/{id}` - Обновление информации об авиакомпании
- `DELETE /airlines/{id}` - Удаление авиакомпании

## Запуск сервиса

```bash
./gradlew bootRun
```

или через Docker:

```bash
docker build -t airline-service .
docker run -p 8080:8080 airline-service
