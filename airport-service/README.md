# Airport Service

Микросервис для управления информацией об аэропортах и их операциях.

## Описание

Airport Service отвечает за хранение и управление данными об аэропортах, включая:
- Регистрацию новых аэропортов
- Управление информацией о существующих аэропортах
- Получение списка всех аэропортов
- Удаление аэропортов

## Основные функции

- Создание, чтение, обновление и удаление записей об аэропортах
- Валидация данных аэропортов
- Интеграция с другими сервисами через REST API

## Технологии

- Spring Boot
- Spring Data JPA
- PostgreSQL
- Docker

## API Endpoints

- `GET /airports` - Получение списка всех аэропортов
- `GET /airports/{id}` - Получение информации об аэропорте по ID
- `POST /airports` - Создание нового аэропорта
- `PUT /airports/{id}` - Обновление информации об аэропорте
- `DELETE /airports/{id}` - Удаление аэропорта

## Запуск сервиса

```bash
./gradlew bootRun
```

или через Docker:

```bash
docker build -t airport-service .
docker run -p 8080:8080 airport-service
