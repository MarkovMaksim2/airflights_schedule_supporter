# Eureka Server

Сервер обнаружения сервисов для микросервисной архитектуры.

## Описание

Eureka Server предоставляет возможность автоматической регистрации и обнаружения микросервисов в системе. Все микросервисы регистрируются в Eureka при запуске и могут находить другие сервисы через него.

## Основные функции

- Регистрация микросервисов при запуске
- Обнаружение сервисов по имени
- Мониторинг состояния сервисов
- Балансировка нагрузки между экземплярами сервисов

## Технологии

- Spring Boot
- Spring Cloud Netflix Eureka Server
- Docker

## Запуск сервиса

```bash
./gradlew bootRun
```

или через Docker:

```bash
docker build -t eureka-server .
docker run -p 8761:8761 eureka-server
```

## Доступ к интерфейсу

После запуска интерфейс Eureka Server будет доступен по адресу:
http://localhost:8761
