# Flight Service

Микросервис для управления расписанием рейсов, их статусами и деталями.

## Описание

Flight Service отвечает за контроль расписания рейсов, включая:
- Создание новых рейсов
- Управление информацией о существующих рейсах
- Получение списка рейсов
- Обновление статусов рейсов (запланирован, задержан, отменён)
- Удаление рейсов

## Основные функции

- Создание, чтение, обновление и удаление записей о рейсах
- Валидация данных рейсов
- Интеграция с другими сервисами через REST API
- Реактивное программирование с использованием Reactor и Spring Data JPA

## Технологии

- Spring Boot
- Spring WebFlux
- Reactor
- Spring Data JPA
- PostgreSQL
- Docker

## API Endpoints

- `GET /flights` - Получение списка всех рейсов
- `GET /flights/{id}` - Получение информации о рейсе по ID
- `POST /flights` - Создание нового рейса
- `PUT /flights/{id}` - Обновление информации о рейсе
- `DELETE /flights/{id}` - Удаление рейса

## Запуск сервиса

```bash
./gradlew bootRun
```

или через Docker:

```bash
docker build -t flight-service .
docker run -p 8080:8080 flight-service
