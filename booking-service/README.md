# Booking Service

Микросервис для управления бронированиями пассажиров и резервированиями.

## Описание

Booking Service отвечает за управление бронированиями рейсов пассажирами, включая:
- Создание новых бронирований
- Управление существующими бронированиями
- Получение списка бронирований
- Отмена бронирований

## Основные функции

- Создание, чтение, обновление и удаление бронирований
- Валидация данных бронирований
- Интеграция с другими сервисами через REST API
- Реактивное программирование с использованием Reactor и R2DBC

## Технологии

- Spring Boot
- Spring WebFlux
- Reactor
- R2DBC
- PostgreSQL
- Docker

## API Endpoints

- `GET /bookings` - Получение списка всех бронирований
- `GET /bookings/{id}` - Получение информации о бронировании по ID
- `POST /bookings` - Создание нового бронирования
- `PUT /bookings/{id}` - Обновление информации о бронировании
- `DELETE /bookings/{id}` - Отмена бронирования

## Запуск сервиса

```bash
./gradlew bootRun
```

или через Docker:

```bash
docker build -t booking-service .
docker run -p 8080:8080 booking-service
