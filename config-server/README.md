# Config Server

Сервер централизованной конфигурации для микросервисов.

## Описание

Config Server предоставляет централизованное управление конфигурацией для всех микросервисов в системе. Он загружает конфигурационные файлы из внешнего хранилища и предоставляет их микросервисам по запросу.

## Основные функции

- Централизованное хранение конфигураций всех микросервисов
- Динамическая загрузка конфигураций
- Поддержка различных профилей окружения (dev, test, prod)
- Шифрование и дешифрование конфиденциальных данных

## Технологии

- Spring Boot
- Spring Cloud Config Server
- Git (для хранения конфигурационных файлов)
- Docker

## Запуск сервиса

```bash
./gradlew bootRun
```

или через Docker:

```bash
docker build -t config-server .
docker run -p 8888:8888 config-server
