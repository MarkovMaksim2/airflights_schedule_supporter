# Passenger Service

Микросервис для управления профилями пассажиров и персональной информацией.

## Описание

Passenger Service отвечает за хранение и управление данными пассажиров, включая:
- Регистрацию новых пассажиров
- Управление информацией о существующих пассажирах
- Получение списка всех пассажиров
- Удаление пассажиров

## Основные функции

- Создание, чтение, обновление и удаление записей о пассажирах
- Валидация персональных данных пассажиров
- Интеграция с другими сервисами через REST API

## Технологии

- Spring Boot
- Spring Data JPA
- PostgreSQL
- Docker

## API Endpoints

- `GET /passengers` - Получение списка всех пассажиров
- `GET /passengers/{id}` - Получение информации о пассажире по ID
- `POST /passengers` - Создание нового пассажира
- `PUT /passengers/{id}` - Обновление информации о пассажире
- `DELETE /passengers/{id}` - Удаление пассажира

## Запуск сервиса

```bash
./gradlew bootRun
```

или через Docker:

```bash
docker build -t passenger-service .
docker run -p 8080:8080 passenger-service
