# Restricted Zone Service

Микросервис для управления зонами ограничения полетов и уведомлениями.

## Описание

Restricted Zone Service отвечает за обработку зон ограничения полетов и автоматическое уведомление авиакомпаний и пассажиров о перекрытиях, включая:
- Создание новых зон ограничения
- Управление информацией о существующих зонах ограничения
- Получение списка всех зон ограничения
- Удаление зон ограничения
- Автоматическую корректировку расписания рейсов при наложении ограничений

## Основные функции

- Создание, чтение, обновление и удаление записей о зонах ограничения
- Валидация данных зон ограничения
- Интеграция с другими сервисами через REST API
- Автоматическая корректировка расписания рейсов при введении ограничений

## Технологии

- Spring Boot
- Spring Data JPA
- PostgreSQL
- Docker

## API Endpoints

- `GET /restricted-zones` - Получение списка всех зон ограничения
- `GET /restricted-zones/{id}` - Получение информации о зоне ограничения по ID
- `POST /restricted-zones` - Создание новой зоны ограничения
- `PUT /restricted-zones/{id}` - Обновление информации о зоне ограничения
- `DELETE /restricted-zones/{id}` - Удаление зоны ограничения

## Запуск сервиса

```bash
./gradlew bootRun
```

или через Docker:
